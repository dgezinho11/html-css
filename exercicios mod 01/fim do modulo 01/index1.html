Exemplo de código em Python

(Sem - Mono-espaçado)

num = int(input('digite um número')) if num % 2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é ímpar') print('FIM DO PROGRAMA') 
     
(Com - Mono-espaçado)

      num = int(input('digite um número'))
      if num % 2 == 0:
          print(f'O número {num} é par')
      else:
          print(f'O número {num} é ímpar')
      print('FIM DO PROGRAMA')

Citações

"Citações" Um trecho de texto escrito ou dito por outra pessoa, que vai ilustrar perfeitamente algo que você quer explicar usando "ASPAS".

Existe outra forma de criar "Citação em HTML" com a tag <q> (do inglês quote, que significa Citar) o texto que estiver entre <q> e </q> já vai receber automaticamente as aspas.

Também podemos criar citaçães (em bloco) e que tenham um parágrafo só pra si. Neste caso, colocaremos tudo dentro de <blockquote></blockquote>. O texto ganha um recuo automaticamente. Podemos também colocar um link para o texto original, usando parâmetro "cite" dentro da tag.
